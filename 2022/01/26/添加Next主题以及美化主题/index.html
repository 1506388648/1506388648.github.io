在我们刚搭建好的博客中，添加入Next主题，并且对Next主题进行美化。

1.Next主题

Next主题官方使用文档：http://theme-next.iissnan.com/

1.1安装Next主题

在博客主目录下右键选择GitBrash输入：

git clone https://github.com/theme-next/hexo-theme-next themes/next

然后在设置站点配置文件_config.yml设置theme类型为Next：

1.2 配置Next主题

1.2.1 选择主题风格

Next主题有四种风格，在博客主目录路径theme/next/_config.yml,选择Muse、Mist、Pisces、Gemini其中之一。我选择的是最后一种主题风格：Gemini
